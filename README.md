# DevOps Copilot AI

An AI-powered DevOps Incident Analysis platform that helps engineers quickly identify root causes, assess severity, generate remediation steps, and create actionable runbooks from DevOps logs.

## Features

### AI-Powered Log Analysis

* Upload DevOps log files
* Supports Jenkins, Kubernetes, Docker, Linux, and AWS logs
* Automated root cause detection

### Incident Intelligence

* Severity Classification (High, Medium, Low)
* Confidence Scoring
* Evidence Extraction
* Suggested Fix Generation
* Prevention Recommendations

### AI Runbook Generation

* Step-by-step remediation workflow
* Incident-specific troubleshooting guidance
* Recovery and investigation procedures

### Command Recommendations

* Context-aware troubleshooting commands
* Platform-specific recommendations

### Reporting

* Export Incident Reports as PDF
* Professional incident documentation
* Downloadable analysis reports

### User Experience

* Modern dashboard UI
* Empty state guidance
* Real-time loading indicators
* Analysis progress tracking
* Responsive design

## Tech Stack

### Frontend

* React.js
* Axios
* jsPDF

### Backend

* FastAPI
* Python

### AI

* Groq LLM

## Workflow

1. Upload a DevOps log file
2. AI analyzes the incident
3. Root cause is identified
4. Severity and confidence are calculated
5. Evidence is extracted
6. Suggested fixes are generated
7. AI runbook is created
8. Recovery commands are recommended
9. Incident report can be exported as PDF
